# Final Project Personal Website

A personal website built for CS-563.

Deployed: https://web.cecs.pdx.edu/~bryc3/

## Running Locally

This is a static site with no server. You can simply run something like:

```bash
python3 -m http.server "$PORT" --directory "$REPO-DIR" &
```

where `PORT` is any available port number to serve it on, and `REPO-DIR` is the name of the directory with this repository.
